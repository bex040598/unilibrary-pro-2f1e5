# ATMU Smart UniLibrary

This repository hosts a premium electronic library platform prototype for Axborot texnologiyalari va menejment universiteti.

## Workspace layout

- `frontend/`: Vite + React + TypeScript client with multilingual routing and dashboard flows.
- `backend/`: FastAPI + SQLAlchemy service with authentication, workflows, seed data, and Swagger docs.
- `docs/`: Architecture, API, database, and deployment notes.

## Delivery rules

- Frontend visual language should feel like a premium academic portal, not a generic landing page.
- Backend endpoints should prefer consistent JSON envelopes and role-aware access checks.
- Seed data must preserve the documented demo accounts.
